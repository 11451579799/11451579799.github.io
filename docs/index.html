如何投稿？

在GitHub上提交issue即可发布文章

  1. 访问本站的GitHub仓库: 11451579799/11451579799.github.io
  2. 点击右上角的 "Issues" 选项
  3. 点击 "New issue" 创建新issue
  4. 填写文章标题和内容，添加适当的标签
  5. 提交后等待审核，审核通过后将自动发布
