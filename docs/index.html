投稿指南

想要分享你的文章？按照以下步骤即可投稿

Step 1

前往 GitHub 仓库

Step 2

创建新的 Issue

Step 3

提交文章内容

详细步骤

1

访问 GitHub 仓库

点击下方按钮前往项目的 GitHub 仓库

打开 GitHub Issues
2

创建新的 Issue

点击 "New Issue" 按钮，填写文章标题作为 Issue 标题

3

编写文章内容

在 Issue 描述中使用 Markdown 格式编写你的文章内容

4

添加标签

为文章添加合适的标签，如：documentation、enhancement、question 等

5

提交并等待

提交 Issue 后，GMEEK 系统会自动将其转换为博客文章

小提示

  • • 支持所有 Markdown 语法，包括代码块、图片、链接等
  • • 图片建议上传到图床服务，如 GitHub、Imgur 等
  • • 文章标题建议简洁明了，便于读者理解
  • • 合理使用标签有助于文章分类和检索
